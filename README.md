# MAFRA PROGETTO
proyectito de robotcito

OBIETTIVO: realizzare un piccolo robot interattivo programmato con l'ESP32 che si muove tramite un app.

OBIETTIVI:
1) Organizzazione del materiale (03/03/26)
🔧 MATERIALI:
- Filo di rame 1 mm
- Filo di rame 0,7–0,8 mm
- ESP32-C3
- Driver motori Mini L298N
- Display OLED 0.96"
- 2× motorini DC con riduttore
- 2× ingranaggi in acciaio (64 denti)
- Batteria Li-Po 3.7V
- Modulo ricarica USB-C (4.2V)
- Interruttore (switch)

🛠️ STRUMENTI:
- Saldatore
- Stagno + flussante
- Pinze / tronchesine
- Pompa dissaldante
